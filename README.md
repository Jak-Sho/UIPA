# 📂 관내출장복명서 -> 액셀 자동화 시스템 (UIPA)

관내출장복명서(PDF)에서 데이터를 추출하여 엑셀 보고서를 자동으로 생성하는 파이썬 도구입니다.

## ✨ 주요 기능
* **데이터 추출:** PDF 파일 내의 출장지, 날짜, 요일 등을 정규식으로 정확히 파싱
* **엑셀 자동화:** 추출된 데이터를 미리 정의된 엑셀 양식에 맞게 기입
* **중복 방지:** 파일명 규칙을 기반으로 이미 처리된 데이터는 제외

## 🛠 실행 방법
1. 바탕화면에 새로운 폴더 생성 -> 내부에 exe 파일을 실행합니다.
2. 새로 생성되는 PDF 폴더에 해당하는 관내출장복명서들을 넣습니다.
2. 프로그램 실행을 하여 확인 후, 엔터를 눌러 작업을 완료합니다.
3. 생성된 엑셀 파일을 확인합니다.

## 📝 참고 사항
* 파이썬 3.x 환경에서 개발되었습니다.
* 필수 라이브러리: `pdfplumber`, `pandas`, `openpyxl` 등
